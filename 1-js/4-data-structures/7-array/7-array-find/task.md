# Поиск в массиве

Создайте функцию `find(array, value)`, которая ищет в массиве `array` значение `value` и возвращает его номер, если найдено, или `-1`, если не найдено.

Например:

```js
arr = ["test", 2, 1.5, false];

find(array, "test"); // 0
find(array, 2); // 1
find(array, 1.5); // 2

find(array, 0); // -1
```
